# 매칭 불가 단어 분류 기준 분석

> graduation 프로젝트 — 한국어-영어 매칭 불가 단어 이미지화 작업 관련 분류 기준 검토

---

## 1. 기존 6개 분류 방식의 문제점

현재 제안된 6개 영역은 **주제별(topic-based) 분류**다:

```
친족어 / 경어 / 의성·의태어 / 색채어 / 식감어 / 정서·관계어
```

이 방식의 문제: 분류 기준이 **의미 도메인(semantic domain)** 이라서,  
각 영역의 "왜 매칭이 안 되는가"라는 핵심 이유가 서로 달라진다.

### 분류 축(axis)이 통일되지 않음

| 카테고리 | 분류 기준 축 |
|---------|------------|
| 친족어 | 의미 도메인 (semantic domain) |
| 경어 | 사회적 기능 (pragmatic function) |
| 의성·의태어 | **형태** (morphophonological form) |
| 색채어 | 의미 도메인 (semantic domain) |
| 식감어 | 의미 도메인 (semantic domain) |
| 정서·관계어 | 의미 도메인 (semantic domain) |

→ **하나의 분류 체계 안에 의미·기능·형태 세 가지 축이 혼재**한다.

## 2. 논리적 모순 3가지

### 모순 1: 식감어 ⊂ 의태어 (부분집합 관계)

피드백 본문에 명시:
> "쫄깃, 바삭 등이 모두 의태어 형태소와 결합"  
> "의태어 연구의 하위범주로 다뤄지는 경우가 많아요"

채완(2003), 박동근(2008)이 정의하는 의태어 체계 안에서 식감어는  
**독립 카테고리가 아니라 하위 클러스터**다.

```
분류의 상호배타성(mutual exclusivity) 위반
→ 바삭 = 의태어 AND 바삭 = 식감어 → 중복 소속
```

### 모순 2: 경어 ∩ 친족어 ≠ ∅ (경계 겹침)

Sohn(1999)의 경어 체계와 Kim(1969)의 친족어 성분분석을 같이 놓으면:

- `아버지` → 친족어 ✓
- `아버님` → 친족어 ✓ **이면서** 경어 ✓

Koh(2002)의 호칭어 연구도 동일 문제를 다룬다.  
친족어가 경어화되는 현상 자체가 두 카테고리가 독립적이지 않다는 증거.

```
분류의 상호배타성 재위반
→ 아버님 = 친족어 AND 아버님 = 경어
```

### 모순 3: 의태어와 경어의 분류 기준이 나머지와 다름

| 카테고리 | 실제 분류 근거 | 인용 논문의 접근법 |
|---------|-------------|----------------|
| 의태어 | **형태론적 체계**의 유무 | Imai & Kita: 음성상징 **시스템** 분석 |
| 경어 | **문법적 인코딩**의 유무 | Sohn(1999): 문법 구조 챕터에서 다룸 |
| 친족어 | **성분 경계** 차이 | Kim(1969): componential analysis |
| 색채어 | **범주 경계** 차이 | Berlin & Kay(1969): 경계 비교 |
| 정서어 | **개념 자체**의 유무 | Wierzbicka/Yoon: NSM 원시개념 분석 |

→ 논문들이 실제로 분석하는 "매칭 안 되는 이유"가 카테고리마다 다르다.  
6개가 같은 이유로 매칭 불가한 것처럼 병렬 나열되면 논리가 어긋난다.

## 3. 논문 기반 재분류: 번역불가 원인(reason for untranslatability) 기준

### Type A: 범주 경계 불일치 (Category Boundary Mismatch)
> **근거 논문**: Berlin & Kay(1969), Kim(1969) — componential analysis

영어에도 개념은 있지만 **경계선이 다르게 그어진** 경우

```
색채어: 파랗다/푸르다 → blue/green 경계가 다름
친족어: 형/오빠/남동생 → brother 하나로 통합됨
```

### Type B: 체계적 구조 부재 (Systemic Structural Absence)
> **근거 논문**: Sohn(1999), Brown(2011), 채완(2003), Imai & Kita

영어에 해당 **표현 시스템 자체가 없는** 경우.  
개별 단어 문제가 아니라 체계 문제.

```
경어: 사회관계가 문법에 인코딩되는 체계 없음
의태어: 음성상징 형태소 생성 체계 없음
식감어: 의태어 체계의 하위 클러스터 (독립 영역 X)
```

### Type C: 개념적 공백 (Conceptual Lacuna)
> **근거 논문**: Wierzbicka(1997), Yoon(2003), Choi et al.(1993)

영어에 개념 자체가 **존재하지 않는** 경우

```
정, 한, 눈치, 정서·관계어 전반
```

## 4. 재분류 후 논리 검토

| 항목 | 결과 |
|-----|------|
| 상호배타성 | ✅ 식감어가 Type B로 흡수 → 중복 소속 해소 |
| 분류 기준 일관성 | ✅ 세 유형 모두 "매칭 불가 원인"으로 통일 |
| 논문 정합성 | ✅ 각 유형이 인용 논문의 분석 방법론과 1:1 대응 |
| 친족어·경어 겹침 | ✅ A형(경계)과 B형(구조)으로 명확히 분리 |

## 5. 시각화 전략과의 연결

| 유형 | 시각화 방식 | 적합한 컴포넌트 |
|-----|-----------|--------------|
| Type A (경계) | 오버랩/스펙트럼, Venn | 경계 영역 표현 |
| Type B (체계) | 관계망, 매트릭스, 트리 | **Node graph 핵심** |
| Type C (개념) | 방사형, 맥락 클라우드 | 단독 노드 강조 |
